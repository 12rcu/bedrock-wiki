# bedrock-wiki-vite

Welcome! You've found the rewrite of the Bedrock Wiki!

## View Website

The website is still in progress, but you can [view it here](https://bedrock-oss.github.io/bedrock-wiki-vite/)

## How to run:

-   `npm install`
-   `npm run dev`
